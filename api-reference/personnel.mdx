<Info>
  **Server-exports.** Kaldes fra `server_script`. Se
  [tillidsmodellen](/api-reference/introduction#tillidsmodellen).
</Info>

Begge exports her tager en **ESX identifier**, ikke en `src` — så de også virker
for betjente der er offline.

```lua server.lua
local officer = exports.va_policemdt:GetOfficer(src)
if not officer then return end

local sheet = exports.va_policemdt:GetTimesheet(officer.identifier)
```

## GetTimesheet

Timer, vagter og vagtstatus.

<ParamField path="identifier" type="string" required>
  Betjentens ESX-identifier.
</ParamField>

<ResponseField name="returns" type="table">
  <Expandable title="felter">
    <ResponseField name="hoursMonth" type="number">
      Timer i indeværende måned.
    </ResponseField>
    <ResponseField name="hoursPrevMonth" type="number">
      Timer i forrige måned — til sammenligning.
    </ResponseField>
    <ResponseField name="week" type="number">
      Timer i indeværende uge.
    </ResponseField>
    <ResponseField name="days" type="table[]">
      Timer fordelt pr. dag over de seneste 14 dage.
    </ResponseField>
    <ResponseField name="shifts" type="table[]">
      De seneste vagter.
    </ResponseField>
    <ResponseField name="avgShift" type="number">
      Gennemsnitlig vagtlængde.
    </ResponseField>
    <ResponseField name="overtime" type="number">
      Overarbejde.
    </ResponseField>
    <ResponseField name="onShift" type="boolean">
      Er betjenten på vagt lige nu?
    </ResponseField>
  </Expandable>
</ResponseField>

```lua server.lua
local sheet = exports.va_policemdt:GetTimesheet(identifier)

if sheet.onShift then
    print(('På vagt — %d timer denne måned'):format(sheet.hoursMonth))
end
```

<Note>
  Kaldet lukker den igangværende vagt af først, så `hoursMonth` tæller også den
  tid betjenten har været på vagt *nu*. Tallet er derfor altid aktuelt.
</Note>

## GetCertificates

Tildelte certifikater og diplomer.

<ParamField path="identifier" type="string" required>
  Betjentens ESX-identifier.
</ParamField>

<ResponseField name="returns" type="table[]">
  <Expandable title="certifikat">
    <ResponseField name="id" type="number" />
    <ResponseField name="name" type="string">
      Fx `Indsatsleder`.
    </ResponseField>
    <ResponseField name="short" type="string">
      Kort form til badges.
    </ResponseField>
    <ResponseField name="description" type="string" />
    <ResponseField name="kind" type="string">
      `certifikat` eller `diplom`.
    </ResponseField>
    <ResponseField name="color" type="string">
      Farvenøgle til visning.
    </ResponseField>
    <ResponseField name="note" type="string">
      Fritekst fra den der tildelte.
    </ResponseField>
    <ResponseField name="awardedBy" type="string">
      Hvem der tildelte det.
    </ResponseField>
    <ResponseField name="awardedAt" type="number">
      Unix-tidsstempel.
    </ResponseField>
  </Expandable>
</ResponseField>

```lua server.lua
local certs = exports.va_policemdt:GetCertificates(identifier)

for _, c in ipairs(certs) do
    if c.short == 'IL' then
        -- betjenten er indsatsleder
    end
end
```

<Tip>
  Certifikater er en pæn måde at gate dine egne features på uden at røre
  rangsystemet: giv `Helikopterpilot` som certifikat, og lad dit
  helikopterscript slå op her.
</Tip>

## Et komplet eksempel

Lås en helikopter til betjente med det rette certifikat.

```lua server.lua
RegisterNetEvent('minhelikopter:udlever', function()
    local src = source

    if not exports.va_policemdt:IsPolice(src) then return end

    local officer = exports.va_policemdt:GetOfficer(src)
    if not officer then return end

    local certs = exports.va_policemdt:GetCertificates(officer.identifier)

    for _, c in ipairs(certs) do
        if c.name == 'Helikopterpilot' then
            return TriggerClientEvent('minhelikopter:spawn', src)
        end
    end

    TriggerClientEvent('chat:addMessage', src, {
        args = { 'Hangar', 'Du mangler certifikat som helikopterpilot.' },
    })
end)
```
