## Forudsætninger

- En kørende FiveM-server med **ESX** og **`va_policemdt`** installeret
- Din egen ressource at skrive i — eller opret en ny mappe i `resources/`

<Note>
  Du behøver ikke ændre noget i `va_policemdt` selv. API'et er exports, som enhver anden ressource kan kalde.
</Note>

## Byg det

<Steps>
  <Step title="Erklær afhængigheden">
    Tilføj `va_policemdt` til din `fxmanifest.lua`. Det er ikke teknisk påkrævet — exports slås op når de kaldes, ikke når ressourcen indlæses — men det får serveren til at advare dig, hvis tabletten mangler.

    ```lua fxmanifest.lua
    fx_version 'cerulean'
    game 'gta5'
    
    dependencies {
        'es_extended',
        'va_policemdt',
    }
    
    server_script 'server.lua'
    ```
  </Step>
  <Step title="Skriv serverkoden">
    Alle data-exports ligger på serversiden. Her registrerer vi en kommando, som enhver spiller kan køre — så det første vi gør, er at kontrollere hvem de er.

    ```lua server.lua
    RegisterCommand('testkald', function(source)
        local src = source
    
        -- Stol aldrig på klienten. Serveren afgør hvem spilleren er.
        if not exports.va_policemdt:IsMdtAuthed(src) then
            return print(('%s er ikke logget ind i MDT\'et'):format(src))
        end
    
        local call = exports.va_policemdt:CreateCall(
            'Testkald',        -- titel
            'Legion Square',   -- sted
            'mellem',          -- prioritet: hoej | mellem | lav
            195.0,             -- x
            -933.0             -- y
        )
    
        print(('Oprettede %s'):format(call.id))
    
        if call.assigned then
            print(('Patrulje %s rykker ud'):format(call.assigned.plate))
        end
    end, false)
    ```

    <Warning>
      `local src = source` i første linje er ikke pynt. `source` er en global, der bliver overskrevet så snart en anden event-handler kører — læser du den senere i funktionen, kan du ramme en helt anden spiller.
    </Warning>
  </Step>
  <Step title="Prøv det">
    Start din ressource og kør `/testkald` i spillet som en betjent, der er logget ind i tabletten.

    ```text
    ensure min_ressource
    ```

    Opkaldet dukker op på kortet hos alle betjente på vagt. Er en patruljevogn ledig, bliver den tildelt og får dispatch-beskeden i radioen.
  </Step>
</Steps>
